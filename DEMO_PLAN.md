# Демо-платформа обучения официантов

## Цель демо

Сделать красивую мобильную демо-версию обучающей платформы, которую можно показать руководству.

Фокус первого демо: путь сотрудника от **стажера** до **официанта** на основе файла `Stazhirovka I Razvitie Oficianta S Bonusami.pdf`.

Главная идея: сотрудник проходит обучение в игровом формате по карте с зигзагообразной дорожкой, открывает миссии, зарабатывает XP, звездочки, достижения и может тратить звездочки на награды.

## Что должно быть видно в демо

- [x] Сотрудник видит свой путь от стажера до официанта.
- [x] Путь отображается как игровая карта в стиле Candy Crush.
- [x] Миссии открываются постепенно.
- [x] За обучение начисляются XP.
- [x] За хорошие результаты начисляются звездочки.
- [x] Есть достижения и бейджи.
- [x] Есть магазин наград за звездочки.
- [x] Наставник может оценить практику стажера.
- [x] После финального испытания сотрудник получает статус "Официант".

## XP и звездочки

### XP

XP показывает общий опыт сотрудника.

- [x] Прошел урок: `+10 XP`.
- [x] Прошел тест: `+15 XP`.
- [x] Сдал сложный тест: `+20 XP`.
- [x] Прошел финальный экзамен: `+50 XP`.

### Звездочки

Звездочки являются внутренней валютой.

- [x] Сдал тест: `+1 звездочка`.
- [ ] Сдал тест на 100%: `+3 звездочки`.
- [x] Наставник поставил 5/5: `+2 звездочки`.
- [ ] Нет опозданий 3 дня: `+3 звездочки`.
- [x] Помог команде: `+1 звездочка`.
- [x] Финальный допуск: `+10 звездочек`.
