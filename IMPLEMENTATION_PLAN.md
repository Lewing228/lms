# План реализации платформы обучения официанта

## Цель продукта

Сделать мобильную обучающую платформу, где сотрудник проходит путь от **стажера** до **официанта** в игровом формате:

- обучение по шагам;
- карта уровней;
- контроль наставником;
- тесты;
- XP и звездочки;
- достижения;
- магазин наград;
- условия допуска;
- прозрачный прогресс для руководителя.

Основа: `Stazhirovka I Razvitie Oficianta S Bonusami.pdf`.

## Текущий фокус

Первая рабочая версия охватывает только путь:

**Стажер -> Официант**

Это соответствует первым трем этапам из PDF:

1. Адаптация новичка.
2. Базовая стажировка.
3. Допуск к самостоятельной работе.

Этапы 4-7 из PDF остаются следующим развитием после запуска первой версии.
